Decrypt Your Text




Project CodeX-b is a web application providing secure client-side text encryption and decryption via shareable URLs made by Dhrrishit Deka. Users input text, set a password (min 8 characters), and the app encrypts the text using AES with a PBKDF2-derived key and a unique salt. This process generates a URL containing the encrypted message, which can be shared for decryption using the same password.
